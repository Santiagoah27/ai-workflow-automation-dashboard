# LinkedIn Featured Copy

## Title Options

- AI Workflow Automation Dashboard
- Full-Stack AI Workflow Automation Demo
- From Manual Processes to AI-Assisted Internal Tools

## Description Option 1: Technical

A full-stack portfolio project built with Next.js, React, TypeScript and .NET 8. It demonstrates structured workflow requests, SQLite persistence, REST API design, layered backend architecture, AI provider abstraction, mock AI-assisted generation, human review and backend tests for core workflow behavior.

## Description Option 2: Business Value

A portfolio project that explores how repetitive internal workflows can be moved from scattered notes, emails and documents into a structured business tool. Users can create requests, generate mock AI-assisted drafts, review and edit outputs, track status and preserve request history.

## Description Option 3: Balanced

AI Workflow Automation Dashboard demonstrates how manual business processes can become structured, traceable and AI-assisted workflows. The project combines a professional internal-tool UI with a .NET backend, SQLite persistence, mock AI generation and a human-in-the-loop review flow.
