# Portfolio Summary

## Short Description

AI Workflow Automation Dashboard is a full-stack portfolio MVP that demonstrates how repetitive business workflows can become structured, traceable and AI-assisted internal tools.

It demonstrates a complete request lifecycle: structured input, mock AI-assisted generation, human review, status tracking, archive flow and persistent request history.

## Medium Description

AI Workflow Automation Dashboard explores how teams can move repetitive internal processes away from scattered notes, emails and documents into a structured workflow system.

The local demo lets users create workflow requests, generate mock AI-assisted drafts, review and edit the output, save a human-reviewed version, archive completed work and keep the request available in history. It was built with Next.js, React, TypeScript and .NET 8, using SQLite persistence, layered backend architecture and an AI provider abstraction prepared for a future real provider.

## Recruiter-Focused Summary

This project demonstrates full-stack engineering across a Next.js and React frontend, a .NET 8 Web API backend, SQLite persistence and a layered architecture with API, Application, Domain and Infrastructure responsibilities.

It highlights practical AI-assisted workflow design through a mock AI provider, provider abstraction, typed API integration and a human-in-the-loop workflow where generated output and reviewed output remain separate. The project also includes backend tests for core workflow behavior and portfolio-ready documentation with real screenshots.

## Client-Focused Summary

This project demonstrates how repetitive business processes can be converted into structured internal tools.

Instead of handling requests through scattered notes, emails or documents, the local demo shows how a business user can capture structured context, generate a draft, review the result, save the reviewed version and keep a traceable history. The AI layer is currently mock-based, and a real provider remains a future improvement through the existing abstraction.
